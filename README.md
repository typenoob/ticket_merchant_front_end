# ticket_merchant

使用flutter开发的车票销售应用，具有管理员和普通用户两个模块，支持多平台。

## 特色设计

- 根据用户的起始站和终点站将整票（从起点站到终点站）拆分成散票。
- 订单支付过程中会触发锁机制，订单支付失败会将订单取消。
- 根据城市间的距离自动计算票价
